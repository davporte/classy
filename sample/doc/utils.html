breakAppart (obj, iter, careAboutStrings, matchCounter, matchA, matchB, hasOccured, matchAOcurred, level, items, stringsActive)
returns a table corresponding to the items descovered between matchA, matchB; by default matchA = '(', matchB = ')'; iter is the string iterated across

Parameters:

  • obj the object making the call
  • iter the data under test, this gets called recursively
  • careAboutStrings true if we care about lookin if the match characters are bound in a string or not, if we care and we are we ignore them when in a string
  • matchCounter counts the number of opens vs closes to ensure we have a complete match
  • matchA the open match character
  • matchB the close match character
  • hasOccured turns true if a pair is found matchA .... matchB
  • matchAOcurred turns true if we have found a matchA only, lets us know to look for matchB
  • level how deep are we in the recursion
  • items a table of the items found
  • stringsActive the current state of strings, are they active i.e. we are in a string, or inactive i.e. we are not in a string

Returns:

    the items brocken appart in a table, c ( an internal counter ), the matchCounter ( should be 0, i.e. complete closure ), has occured true or false

See also:

    stringToTable

Usage:

  • obj:breakAppart ( String, true, 0, '{', '}' ) -- breaks appart tables
  • obj.breakAppart ( Obj, String, true, 0, '(', ')' ) -- breaks appart parenthesis
