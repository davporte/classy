myLogger
a function you can set to do something with logs, if not set then the default logger is called. If myLogger returns true then the default logger will also be called, if false/nil it will not be. If myLogger returns any other value then the default logger is called to display that value. Note that myLogger still obays _LOGGING, _SUBLOGGING settings

Usage:

  • classy.myLogger = function ( log )
-DO SOME STUFF WITH log
end
This does not trigger the standard logger to also log
  • classy.myLogger = function ( log )
DO SOME STUFF WITH log
return true
end
This does trigger the standard logger to log

Global Attributes (Protected)

overloadOperators
operators that a class can be overload, > and >= supported using not > and not >=

Fields:

  • + add
  • - subtract
  • * multiply
  • / divide
  • % mod
  • :- unary minus, note colon on front of -
  • = equals
  • < less than
  • <= less than or equals to
validAttributeDescribers
These values are used to describe Private / Immutable state of an attribute Immutable attrubutes CANNOT change, Private attributes CANNOT be refered to outside the class object methods

Fields:

  • Immutable Immutable, Public same as PublicImmutable
  • Public Mutable, Public same as PublicMutable
  • PublicMutable Mutable, Public same as Public
  • PublicImmutable Immutable, Public same as Immutable
  • Private Mutable, Private same as PrivateMutable
  • PrivateMutable Mutable, Private same as Private
  • PrivateImmutable Immutable, Private
validBaseTypeDescribers
the valid base classes that attributes can be, these are stored in the global table _G and are protected, excluding ClassType (custom) - these are held in classy

Fields:

  • String a string type
  • Number a number type
  • Table a raw table, note this is not valid for a class
  • Bool a boolean type
  • Function a function type
  • ClassType a custom class, created by calling newClass

See also:

    validAttributeDescribers

Usage:

  • attribute = validBaseTypeDescribers (  validAttributeDescribers ) -- attribute defining
  • object = validBaseTypeDescribers ( [{ attribute = value ... attribute = value }] ) -- object creation

External Call (Protected)

callSuperMethod (obj, method, pass)
allows an object inside a classy constructure to call their supers init, or any other method. this CAN NOT be called from outside a classy: constructor method.

Parameters:

  • obj the object wishing to call there supers method
  • method the method name you wish to call
  • pass the ... up to the method

Returns:

    what ever the super init method returned
addToProtectionIn_G (key, value)
adds key in key with a value to the protected _G table also adds this key into the user protected table, this ensures that removeFromProtectionIn_G can only affect user defined values

Parameters:

  • key the name of the key in _G
  • value the value that key will have in _G
removeFromProtectionIn_G (key)
removes key in key with a value from the protected _G table this only happens with user defined values. You can't do this with system protecteed values

Parameters:

  • key the name of the key in _G
