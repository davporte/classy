Module fileHandler

creates a file handler class.

designed for use with the classy object creator

Usage:

    FileHandler = require ( 'fileHandler' )

initMethod (obj, args)
sets the default read behavour flags

Parameters:

  • obj the calling object
  • args optional arguments sent to create the file handler

Returns:

    No return value

Usage:

    myFileHandler = FileHandler ()
setDefaultReadBehavour (obj, param1, param2)
setDefaultReadBehavour can adjust the default corona behavour of one line at a time for a read, it can also force a file close at the end takes to paramters param1, param2, they are not manditory but if there one must be boolean the other a string the boolean sets the closeAfterRead and the string the adjusted file default read method.

Parameters:

  • obj the file hander object
  • param1 the first parameter to check
  • param2 the second parameter to check

Returns:

    No return value

Usage:

    fileHandler:setDefaultReadBehavour ( param1 [, param 2 ] )
openFile (obj, fileDetails, mode)
opens a file

Parameters:

  • obj the file handler object
  • fileDetails is an actual path or a table containing filename/path
  • mode is the mode you want the file to open in, if not specified then this is the defaultReadBehavour "r"

Returns:

    a file handler or nil

See also:

Usage:

    local fhd = fileHandler:openFile ( { fileName = fileName [, path = pathToFile ] } [, mode ])
closeFile (obj, fhd)
closes a file

Parameters:

  • obj the file handler object
  • fhd the file you wish to close

Returns:

    true if success false if failed

See also:

Usage:

    local success = fileHandler:close ()

fileNameBuilder (obj, fileDetails)
builds file name if path table otherwise returns orgional value. You may have to overload this with your own method if not using Corona SDK

Parameters:

  • obj the file handler object
  • fileDetails either a path table or a fileName as a string

Returns:

    a filename format for Corona SDK
