Module stack

creates a stack class.

designed for use with the classy object creator

initMethod (obj, args)
adds the default log levels Error, Warning, Info and Debug and sets their associated default values ON (true), ON (true), OFF (false), OFF (false). also sets the logger to the default output.

Parameters:

  • args optional arguments sent to create the stack

Returns:

    No return value

Usage:

  • myStack = Stack ()
  • myStack = Stack ( { stack = { item1, ...., itemN } } )
pop (obj, howMany)
pops an items off a stack. howMany allows more than one item to be popped.

Parameters:

  • obj the stack object
  • howMany how many items you want off the stack

Returns:

    either a single item. If howMany is set then that number, if available, is returned in a table. With a paramter n = the number of entries

Usage:

    local result = myStack:pop ( [ number ] )

overloaded (obj1, obj2)
overloads + for stacks, adds two stacks together in one stack and drains the other stacks. can also do + just to add new object to stack, in this case obj2 is not a stack object

Parameters:

  • obj1 the first stack object
  • obj2 the second stack object, or another object type

Returns:

    A new stack object

Usage:

    local newStack = oldStack + anotherStack | object [ + ... + [ + anotherStack | object ] ]
